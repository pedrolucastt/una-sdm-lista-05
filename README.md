# ConsumerGitHubUser

Aplicação Console em C# desenvolvida para consumir a API pública do GitHub e exibir informações de um usuário.

## 📌 Objetivo

Consumir o endpoint da API do GitHub e exibir no console:

- Nome
- Empresa
- Localização
- Login

## 🚀 Tecnologias utilizadas

- .NET
- C#
- HttpClient
- System.Text.Json
